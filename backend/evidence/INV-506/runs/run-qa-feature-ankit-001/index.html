Issues Flagged

⚠️ MRDT rate: 2% shown, spec documents 3%
Both API (/so/folio, /so/invoice) and UI (folio dialog, invoice dialog, Invoice Summary sidebar) show MRDT at 2%. Product spec and Known Risk Areas document MRDT as 3%. The math is internally consistent at 2% (119.00 × 0.02 = $2.38). Likely a property-level configuration for Totem Lodge — confirm with dev whether this is intentional.
⚠️ API invoice.currency is empty string instead of "CAD"
GET /so/folio and /so/invoice return "currency": "". The spec implies CAD. UI correctly shows "$" / "CAD" from other fields (unit_price_curr), so no user-visible impact — but the API contract has a gap.
ℹ️ /reservation/:id/folio URL redirects to dashboard
Direct navigation to /reservation/BK_J755DICT/folio redirects to /dashboard. Folio and Invoice are only accessible as modal dialogs from the reservation detail page. This is consistent with the implemented design (modal preview + print), but the URL listed in the product spec's Key Pages section may be outdated.
